# Suicide Risk Level Prediction (ML Project)

##  Project Overview
This project predicts suicide risk level using psychological and demographic features through Machine Learning models.

##  Dataset
Source: Kaggle 
The dataset includes the following features:
- Depression Level
- Anxiety Level
- Stress Level
- Self Harm Story
- Mental Support
- Family Problem
- Relationship Condition
- Age
- Gender
- Academic Performance
- Health Condition

Target Variable:
- Suicide Attempt (Risk Level)

##  Exploratory Data Analysis (EDA)
- Missing value handling
- Correlation heatmap
- Feature importance analysis
- Class distribution check

##  Machine Learning Models Used
- Logistic Regression (Baseline Model)

##  Evaluation Metrics
- Accuracy
- Confusion Matrix
- ROC-AUC Curve
- Classification Report
